第二题

没有一次性过，报了这个错误:
member access within misaligned address 0x000000000031 for type ‘struct ListNode’, which requires 8 byte alignment

由于结构体内存在next指针，而申请结构体空间后同时定义了next指针，此时next指针未指向任何空间，故在测试时可能导致上述错误。
解决方法为：
增加代码使next指针指向空。

node->next=NULL;
